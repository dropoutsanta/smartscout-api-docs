## Plan

| | |
| --- | --- |
| **Price** | **\$999 / month** |
| **Billing** | Monthly, billed in advance |
| **Access** | Full SmartScout tool suite |
| **Transport** | REST (`/api/v1/*`) and MCP |
| **Base URL** | `https://smartscout-mcp.fly.dev` |
| **Limits** | Plan rate limits (e.g. daily request caps); fair-use concurrency |

No per-call metering on this plan beyond your key's rate limits — you pay the flat monthly fee for API access.

## What's included

- Brand search, profiles, history, coverage, and **full brand reports**
- Product search, Buy Box, history, organic ranks, relevancy
- Sponsored search-term / ad win-rate intelligence
- Category tree, market sizing, leaders, and trends
- Built-in playbook tool (`smartscout_how_to_use_me`)
- Dedicated API key (issued privately)

## What's not included

- Contact / email enrichment (this API is Amazon marketplace data only)
- SmartScout UI seats (API access is separate from any SmartScout web login you may already have)

## Using your key

Once you have a key, set it and call the gateway:

```bash
export SMARTSCOUT_API_KEY="ssk_..."

curl -X POST "https://smartscout-mcp.fly.dev/api/v1/call" \
  -H "Authorization: Bearer $SMARTSCOUT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"tool":"smartscout_how_to_use_me","params":{}}'
```

For agents, see [MCP setup](/mcp). For scripts, see the [Quickstart](/quickstart).
