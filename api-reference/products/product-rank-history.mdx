`smartscout_product_rank_history`

Daily history of organic rank shifts across search terms.

## Parameters

| Name | Required | Description |
| ---- | -------- | ----------- |
| `product_id` | Yes | Numeric product ID |
| `marketplace` | No | Default `US` |

## Example

```bash
curl -X POST "https://smartscout-mcp.fly.dev/api/v1/call" \
  -H "Authorization: Bearer $SMARTSCOUT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"tool": "smartscout_product_rank_history", "params": {"product_id": 987654}}'
```
