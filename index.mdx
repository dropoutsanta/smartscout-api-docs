Programmatic access to Amazon marketplace intelligence: brand revenue, product performance, Buy Box competition, organic ranks, ad win rates, and category sizing.

**\$999 / month** flat — unlimited tool calls on your plan. Base URL and API key are issued when you subscribe.

## Your first request

```bash
curl -X POST "$SMARTSCOUT_API_BASE/api/v1/call" \
  -H "Authorization: Bearer $SMARTSCOUT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "tool": "smartscout_search_brands",
    "params": { "name": "OxiClean", "limit": 5 }
  }'
```

## All tools

Every call goes through one endpoint: `POST /api/v1/call` with a tool name and params.

| Tool | Group | What it does |
| ---- | ----- | ------------ |
| `smartscout_how_to_use_me` | Guide | Built-in playbooks and tool overview |
| `smartscout_search_brands` | Brands | Find brands by name or category + revenue |
| `smartscout_get_brand` | Brands | Full brand snapshot (~30 fields) |
| `smartscout_brand_report` | Brands | **All-in-one dossier** (preferred for research) |
| `smartscout_brand_history` | Brands | Weekly revenue / units / price trends |
| `smartscout_brand_history_by_category` | Brands | Revenue mix by subcategory over time |
| `smartscout_brand_coverage` | Brands | 3P seller distribution for a brand |
| `smartscout_brand_tailored_report` | Brands | Pre-generated report freshness check |
| `smartscout_search_products` | Products | Search products by brand, ASIN, title, category |
| `smartscout_product_sellers` | Products | Buy Box sellers for a product |
| `smartscout_product_history` | Products | Weekly product performance time-series |
| `smartscout_product_organic_ranks` | Products | Current organic search ranks |
| `smartscout_product_rank_history` | Products | Daily organic rank changes |
| `smartscout_product_relevancy` | Products | Competing ASINs by relevancy |
| `smartscout_search_terms_for_product` | Ads | Sponsored ad win rates by search term |
| `smartscout_subcategories` | Categories | Top-level category tree (names + IDs) |
| `smartscout_search_subcategories` | Categories | Market sizing with revenue & growth stats |
| `smartscout_subcategory_brands` | Categories | Top brands in a subcategory |
| `smartscout_subcategory_history` | Categories | Category revenue / units trends |
